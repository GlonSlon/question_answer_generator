# Автоматический генератор вопросов и ответов из текстовых файлов

Инструмент на Python, который автоматически генерирует пары «вопрос-ответ» из текстовых документов с использованием методов обработки естественного языка и машинного обучения. Система извлекает ключевую информацию из файлов `.txt` и создает пары «вопрос-ответ», подходящие для обучающих наборов данных или баз знаний.

## Особенности

- **Обработка нескольких файлов**: считывает все файлы `.txt` в текущем каталоге (кроме `dataset_gpt.txt`)
- **Интеллектуальный анализ предложений**: идентифицирует предложения-определения и ключевые понятия
- **Генерация вопросов**: создает релевантные вопросы на основе содержания предложений
- **Ранжирование с использованием машинного обучения**: использует TF-IDF для определения приоритетности важных предложений (при наличии scikit-learn)
- **Дедупликация**: удаляет повторяющиеся вопросы и фильтрует пары низкого качества
- **Форматы вывода**: генерирует как простой текстовый набор данных, так и файл метаданных JSON

## Требования

- Python 3.7+
- Зависимости (необязательно, но рекомендуется):

- `nltk`: улучшенная токенизация предложений

- `spacy`: улучшенное извлечение ключевых терминов (с моделью en_core_web_sm)

- `scikit-learn`: на основе TF-IDF Ранжирование важности предложений

- `tqdm`: Индикаторы выполнения

## Установите необходимые зависимости:
```bash
pip install nltk spacy scikit-learn tqdm
```
## Загрузите модель spaCy для английского языка (если используете spaCy):
```bash
python -m spacy download en_core_web_sm
```
## Использование:
Поместите ваши файлы .txt в ту же директорию, что и скрипт.

```bash
python qa_generator.py
```
## Скрипт выполнит следующие действия:
- Обработает все файлы .txt (кроме dataset_gpt.txt)
- Сгенерирует пары вопрос-ответ
- Сохранит результаты в файлы dataset_gpt.txt и dataset_gpt_meta.json
